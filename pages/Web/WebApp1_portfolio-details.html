A real estate application built using ReactJS!

Project information

  • Category: Web design
  • Client: ASU Company
  • Project date: 01 March, 2023
  • Project URL: Click Here

With this app, users can search for properties by location and filter results by price, type (house or apartment), and number of bedrooms and place. And with a beautiful UI, the experience of finding your dream home has never been easier.

This project covers the fundamentals of ReactJS, including functional components, hooks such as useState, useEffect, and useContext, and the use of React Icons and Routers. Additionally, I've incorporated large amounts of data to ensure that users have access to a wide variety of properties.Plus, the app is fully responsive, so it looks great on any device.
